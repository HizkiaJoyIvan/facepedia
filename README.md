Main feature of the app:
1. Create post
2. Like post
3. Search another users
4. See profile page of another users
5. Follow/unfollow another users
6. Edit photo profile/username

Upcoming feature:
1. Direct chat with friends
2. Event notifications
3. Change language
4. Post filtering
